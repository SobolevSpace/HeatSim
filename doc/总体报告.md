# 问题及背景

在工业生产过程中，我们经常需要对二维热场进行模拟。比如笔记本电脑厂商需要设计出散热更好的平板结构，工厂的建设规划需要保证厂房结构的散热，而这些问题都可以建模成二维热场的模拟。
本项目需要实现一个科学仿真软件，该软件能模拟平面区域温度场的变化，并能根据用户的需要编辑平面，比如：在平面上放置恒温源并设定温度，放置隔热板，设置传热参数等等。后因有余力又增加了类似画图编辑的功能。
